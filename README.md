# CampApp
 **Frontent:**
 1. Install packages :
    ```
    npm install --legacy-peer-deps
    ```
 2. Start server :
    ```
    npm run dev
    ```
    
   **Backend:**
 1. Install packages :
    ```
    npm install 
    ```
2. Create .env file  :
    ```
    LLM_API_KEY= your key
    JWT_SECRET= your key
    ```
3. Start server :
    ```
    npm start
    ```
  at 
    ```
    http://localhost:5173/)
    ```
